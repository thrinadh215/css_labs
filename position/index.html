CSS Position Property Demonstration

Static Position

This is a static positioned box.

The static position is the default. Elements are positioned according to the normal document flow.

Relative Position

This box is relatively positioned.

A relative positioned element is positioned relative to its normal position. Moving it doesn't affect the layout of other elements.

Absolute Position

This box is absolutely positioned.

An absolutely positioned element is positioned relative to its nearest positioned ancestor (other than static) or relative to the initial containing block if none exist.

Fixed Position

This box is fixed positioned.

A fixed positioned element is positioned relative to the browser window and remains in the same place even when scrolling.

Sticky Position

This box is sticky positioned.

A sticky positioned element toggles between relative and fixed, depending on the scroll position.
